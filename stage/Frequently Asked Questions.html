FAQ

1. The plugin doesn't work and when I click on the ! icon I see something telling me I need to upgrade to Hotmail Plus, what's going on?

Microsoft has recently decided to start charging for WebDav access to Hotmail. This will affect all newly created accounts, however some people with older Hotmail accounts will still be able to connect using the plugin as it is taking Microsoft quite some time to convert all the accounts over to their new system. If you are seeing this error then the only way to make the plugin with with your account is to "upgrade" to Hotmail Plus.

2. Can the plugin connect to xxx web email?

The short answer is no.

The long answer is, no. This is because the httpmail plugin relies on a protocol implemented by hotmail. Currently services like Yahoo and Mail.com do not provide the special protocol the plugin requires. It could be possible to write another plugin specifically designed to access these web based email systems, but that is a very big job.
